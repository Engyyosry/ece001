Advantages and disadvantages

advantages of programming languages :
1. Readability: A good, high-level language allows programs to be written in a manner similar to an English language description.
2. Portability: High-level languages, which are essentially machine-independent, should be able to develop portable programs.
3. Generality: Most high-level languages allow writing a wide range of programs, thus reducing the programmer's need to become an expert in many different languages.
4. Briefness: The language must have the ability to implement the algorithm with minimal code. .
5. Error checking: Many high-level languages impose a great deal of error checking at compile and runtime time.
6. Cost: The final cost of a programming language is a function of many of its characteristics.
7. Familiar coding: The language must have a familiar notation that can be understood by most programmers.
8. Modularity: It is recommended that programs be developed in the language as a set of modules translated separately, with appropriate mechanisms to ensure self-consistency between these modules.
9. Widely available: the language must be widely available and it should be possible to provide translators for all major machines.
